# Interactive Language Learning Book App

تطبيق ويب متقدم لتحويل كتب تعلم اللغات إلى منصات تفاعلية رقمية محفوظة الأصلية.

## المميزات الرئيسية

- 📖 **الحفاظ على تخطيط الكتاب الأصلي** - تصميم كامل RTL
- ✍️ **تمارين تفاعلية** - أسئلة، ملء الفراغات، اختيارات متعددة
- 🎵 **دعم صوتي** - ربط ملفات صوتية بالتمارين
- 💾 **حفظ التقدم** - مزامنة سحابية تلقائية
- 🎨 **تظليل وتعليقات** - أدوات تفاعلية مباشرة على الصفحات
- 🤖 **مساعد ذكي** - شرح الدروس والقواعس النحوية والترجمة
- 📊 **إحصائيات التعلم** - تتبع التقدم والإحصائيات

## لغة التصميم

- **اللون الأساسي**: أزرق #2563EB
- **الخلفية**: بيضاء نظيفة
- **الكروت**: حواف مدورة كبيرة (rounded-2xl)
- **الظلال**: خفيفة بدل الخطوط الفاصلة
- **الأيقونات**: outline style بسيطة
- **الدعم**: RTL كامل للعربية والألمانية

## المتطلبات

- Node.js 16+
- npm أو yarn
- MongoDB (محلي أو سحابي)

## التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/cr72000moo-art/interactive-language-book.git
cd interactive-language-book

# تثبيت المتطلبات
npm install

# تشغيل التطبيق
npm run dev
```
